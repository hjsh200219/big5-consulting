# Product Sense -- Big 5 Consulting MCP Server

## 제품 비전

Big 5 성격 심리학을 AI 어시스턴트에 통합하여,
개인과 팀이 데이터 기반으로 더 나은 의사결정을 내리도록 돕는다.

## 핵심 가치

1. **개인화된 인사이트**: 성격 데이터 기반 맞춤형 분석
2. **팀 최적화**: 팀 구성, 역할 배분, 갈등 예방
3. **접근성**: Claude Desktop 통합으로 자연어 기반 상호작용
4. **프라이버시**: 로컬 파일 시스템 저장, 외부 전송 없음

## 사용자 페르소나

### 1. HR 전문가 (Primary)
- **니즈**: 팀 구성 최적화, 갈등 예방, 채용 적합도 평가
- **사용 도구**: analyze_team (composition, roles, conflicts)
- **핵심 시나리오**: 신규 팀원 합류 시 팀 역학 사전 분석

### 2. 커리어 코치 (Primary)
- **니즈**: 개인 강점 기반 커리어 추천, 자기계발 가이드
- **사용 도구**: analyze_individual (career, development, learning_style)
- **핵심 시나리오**: 내담자 성격 프로필 기반 맞춤 코칭

### 3. 팀 리더/매니저 (Secondary)
- **니즈**: 팀 소통 스타일 이해, 협업 효율성 향상
- **사용 도구**: analyze_team (communication, collaboration)
- **핵심 시나리오**: 팀 갈등 발생 시 원인 분석 및 소통 전략

### 4. 자기 탐구 개인 사용자 (Secondary)
- **니즈**: 자신의 성격 특성 이해, 스트레스 관리
- **사용 도구**: manage_survey, analyze_individual (personality, stress_management)
- **핵심 시나리오**: 성격검사 후 종합 프로필 해석

## 제품 원칙

### 과학적 근거
- Big 5 모델은 심리학에서 가장 널리 검증된 성격 이론
- 60문항(전체) / 30문항(간략) 두 가지 버전으로 정밀도-편의성 균형
- 역채점 문항을 통한 응답 신뢰성 확보

### 점진적 경험
1. 설문 시작 -> 5문항씩 제공 (인지 부담 최소화)
2. 진행 상황 추적 가능 (resume/progress)
3. 완료 시 프로필 자동 생성
4. 다양한 관점의 분석 제공 (6종 개인 + 5종 팀 + 3종 관계)

### 데이터 주권
- 모든 데이터가 로컬 (`~/.big5/`)에 저장
- 외부 API 호출 없음
- 사용자가 언제든 데이터 삭제 가능

## 제품 차별점

| 기존 접근 | Big 5 MCP |
|-----------|-----------|
| 웹 기반 설문 -> PDF 리포트 | AI 대화 내에서 자연스러운 설문 + 실시간 분석 |
| 일회성 결과 조회 | 프로필 저장 후 다양한 분석 반복 가능 |
| 개인 분석만 제공 | 팀/관계 분석까지 확장 |
| 외부 서버 전송 필수 | 로컬 저장 (프라이버시) |
